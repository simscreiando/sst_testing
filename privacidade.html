Aviso de Isenção de Responsabilidade (Electronic Arts):
O SimStoria é um aplicativo de suporte (Companion App) criado por fã para fãs, com o objetivo de enriquecer a experiência narrativa do jogo. Este projeto NÃO é afiliado, endossado, patrocinado ou aprovado pela Electronic Arts Inc. (EA). "The Sims", "The Sims 4", "Simoleons" e elementos relacionados são marcas registradas da Electronic Arts Inc. O uso de terminologias e referências ao jogo segue as diretrizes de Política de Conteúdo de Fãs da EA, possuindo caráter estritamente não comercial.

1. Termos de Uso

1.1. Natureza do Serviço

O SimStoria é oferecido gratuitamente, no estado em que se encontra. Por ser um projeto em constante desenvolvimento, não há garantias de que a plataforma estará permanentemente livre de erros, instabilidades ou interrupções técnicas.

1.2. Responsabilidade sobre Dados e Progresso

O usuário é inteiramente responsável pela gestão e segurança de suas informações na plataforma:

  • Armazenamento Local: Caso opte por não conectar uma conta em nuvem, os dados da sua jogabilidade ficam salvos apenas no navegador do seu dispositivo. Limpezas de cache ou formatações apagarão o progresso de forma irreversível.
  • Cópias de Segurança: Recomenda-se que o usuário faça o download manual de seus dados (backups) periodicamente através das ferramentas disponibilizadas na interface do sistema.

1.3. Integração com Chaves de API de Inteligência Artificial

O SimStoria permite o uso opcional de recursos de Inteligência Artificial através do fornecimento de chaves de API próprias do usuário (como OpenAI, Anthropic ou similares). O SimStoria não se responsabiliza por quaisquer custos, cobranças, limites estourados ou bloqueios gerados nas contas externas do usuário junto a esses provedores.

1.4. Doações e Apoio ao Projeto

Contribuições financeiras realizadas voluntariamente por meio de PIX ou plataformas de apoio servem exclusivamente para ajudar nos custos de manutenção técnica da plataforma e incentivo à desenvolvedora. Elas não configuram a compra de produtos, assinatura de serviços com obrigações contratuais ou garantia de recursos exclusivos, não sendo passíveis de reembolso.

2. Política de Privacidade

2.1. Filosofia de Armazenamento "Client-Side"

O SimStoria preza pela soberania dos seus dados. Nós não possuímos servidores de banco de dados centralizados para guardar suas histórias, árvores genealógicas ou perfis. Tudo permanece sob o seu controle direto no seu dispositivo ou na sua conta de nuvem privada.

2.2. Integração e Uso de Dados do Google Drive

Ao ativar voluntariamente a sincronização com o Google Drive, o aplicativo solicitará permissões específicas de acesso (como a criação de arquivos próprios e manipulação da pasta de dados do app - appDataFolder):

  • O SimStoria acessará apenas e exclusivamente os arquivos criados pelo próprio aplicativo para fins de salvar o progresso do usuário (gerar backups e carregar dados de famílias).
  • O aplicativo NÃO possui autorização, não lê, não coleta e não modifica nenhum outro documento, foto ou arquivo pessoal existente no Google Drive do usuário.
  • O tráfego de dados de autenticação ocorre de forma direta e segura entre o navegador do usuário e os servidores do Google.
  • O uso e a transferência de informações recebidas das APIs do Google para qualquer outro aplicativo pelo SimStoria aderem à Política de Dados de Usuário de Serviços de API do Google, incluindo os requisitos de Uso Limitado.

2.3. Tratamento de Chaves de API e Dados de IA

As chaves de API inseridas no sistema são armazenadas localmente de forma criptografada ou direta no armazenamento do seu navegador (LocalStorage). Elas nunca são enviadas para servidores do SimStoria. Os dados de texto enviados para geração de histórias trafegam diretamente para os servidores das empresas de IA escolhidas pelo usuário.

2.4. Formulários de Feedback

Informações inseridas voluntariamente em campos de contato ou feedback (como nome/apelido e e-mail) serão utilizadas unicamente para comunicação de suporte técnico e melhorias no sistema, jamais sendo comercializadas ou compartilhadas com terceiros.
